HCRprocess
===================================

HCRprocess (`GitHub <https://github.com/economolab/HCRprocess>`_) is an `HCR-mFISH <https://journals.biologists.com/dev/article/145/12/dev165753/48466/Third-generation-in-situ-hybridization-chain>`_ image processing pipeline written in MATLAB and Python 3 by Jack Vincent. It is composed of components developed by Jack Vincent, Michael Economo, and Will Cunningham.
